Micropayment is a viable solution to ads by monetizing your content via a paywall.

== Description ==

Introducing a complete multi merchant paywall, that brings users and website publishers together in one cohesive environment.

When you activate the micropayment.io plugin and or gateway you open up your website to charge micro payments to the tune of $0.01.

Simply add the shortcodes around the text, audio & video players, or download links you want to monetize, it couldn't be easier.

Site visitors can load their Micropayment.io Chrome extension with KITs and use them on any website that has the Micropayment.io plugin enabled.

[youtube https://youtu.be/sorJFWal21A]

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/micropaymentio` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Use the Settings->Micropayment screen to configure the plugin.
4. Setup plugin via the configuration wizard.
5. Create an account on [BillingFox](https://billingfox.com) to get your free API key.
6. Mark the content you want to protect via the [micropay] [/micropay] shortcodes.
7. Build your business and add another revenue stream and profit.
